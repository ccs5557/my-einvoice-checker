# Malaysia e-Invoice Compliance Checker

一个用于快速评估马来西亚 e-Invoice 合规风险的互动小测验（3 个问题）。适合作为营销引流页/Lead Magnet 的演示项目，支持中英双语切换，并在结果页给出风险等级与建议。

## 主要功能
- 3 步问答流程，输出低/中/高风险评估与建议
- 支持中/英文语言切换
- 精美结果页与 CTA（WhatsApp 联系）示例
- 适合嵌入营销落地页或产品演示

## 技术栈
- React + TypeScript + Vite
- Tailwind CSS
- framer-motion（动效）
- lucide-react（图标）
